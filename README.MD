## ğŸ”¹ Tips for VS Code

* Ensure `"emmet.triggerExpansionOnTab": true` is set in `settings.json`.
* Use `.html` file type for full support.
* For Emmet in plaintext, add:

  ```json
  "emmet.includeLanguages": {
    "plaintext": "html"
  }
  ```
